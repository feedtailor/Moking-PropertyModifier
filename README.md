Moking-PropertyModifier
=======================

A property modifier for Mock object.


Install
-----

feedtailor/mocking-property-modifier を composer.json に追加して、 `composer install` を実行します。

```json
{
  "require": {
    "feedtailor/mocking-property-modifier": "dev-master"
  }
}
```
